## Eigen venv-experiment (deployment)

# venv-deploy-experiment

## Beschrijving

Dit experiment demonstreert hoe een Python-applicatie reproduceerbaar kan worden uitgerold (deployment) met behulp van een virtual environment (venv).  
Er wordt een eenvoudige Python-applicatie ontwikkeld die een HTTP-request uitvoert naar een publieke API.

De focus van het experiment ligt niet op de functionaliteit van de applicatie, maar op het correct beheren en vastleggen van afhankelijkheden met pip en requirements.txt.

## Doel

Het doel van dit experiment is om aan te tonen dat:

- een Python virtual environment geïsoleerd kan worden aangemaakt
- projectafhankelijkheden correct geïnstalleerd en vastgelegd kunnen worden
- een applicatie reproduceerbaar kan worden uitgevoerd in een nieuwe, schone omgeving
- een venv niet wordt gedeployed, maar opnieuw wordt opgebouwd op de target machine
