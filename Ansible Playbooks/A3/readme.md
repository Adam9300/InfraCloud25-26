# Ansible Nginx Webserver Experiment

## Beschrijving
Aantonen dat Ansible gebruikt kan worden
om verschillende soorten servers automatisch te installeren en configureren.
In dit experiment wordt Nginx gebruikt in plaats van Apache.

## Doel
- Automatische installatie van serversoftware
- Aanpassen van configuratiebestanden
- Gebruik van variabelen voor flexibiliteit
- Uitrollen van een custom webpagina
