## 4.9.2 Lab - Integrate a REST API in a Python Application  

## Beschrijving
In deze lab wordt een REST API geïntegreerd in een Python applicatie.
De applicatie haalt data op via HTTP requests en verwerkt de response.

## Functionaliteit
- Versturen van HTTP GET requests
- Ontvangen en parsen van JSON responses
- Weergeven van API data in de terminal

## Bestanden
- graphopper file – Python applicatie
- README.md – Lab uitleg

## Doel
Inzicht krijgen in:
- de werking van REST APIs in een Python app
- data van API weergeven in een terminal
