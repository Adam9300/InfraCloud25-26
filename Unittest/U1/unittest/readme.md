#  3.5.7 Lab – Create a Python Unit Test

## Beschrijving
In deze lab wordt Python unittest gebruikt om een functie te testen die
recursief zoekt in een JSON-object.

## Bestanden
- recursive_json_search.py – Bevat de functie json_search()
- test_data.py – JSON testdata en zoekkeys
- test_json_search.py – Unit tests voor de functie

## Doel 
- Unittest aanmaken om een functie te testen
