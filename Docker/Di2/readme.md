## Image Experiment Web Service (Flask + Docker)

Kleine webservice om een afbeelding te uploaden en een eenvoudige bewerking toe te passen.
In dit experiment: grayscale (zwart-wit). Daarna krijg je origineel + resultaat te zien.

## Doel
- webservice aanmaken om een image up te loaden 
- via curl of docker webpagina open krijgen

## Features
- Upload een afbeelding (jpg/png)
- Server verwerkt de upload en maakt een grayscale-versie
- Resultaat wordt getoond in de browser
