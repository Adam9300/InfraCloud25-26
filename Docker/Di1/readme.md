## 6.2.7 Lab - Build a Sample Web App in a Docker Container  

## Beschrijving
In deze lab bouw je een eenvoudige Python Flask webapp
en deploy je deze in een Docker container met een bash script.

## Bestanden
- sample_app.py – Flask webapp
- sample-app.sh – Bash script om Docker image te bouwen en runnen
- user-input – Simpel bash script (input/output)

## Mappen
- templates/index.html – HTML pagina
- static/style.css – CSS styling
