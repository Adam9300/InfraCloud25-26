# Eigen REST-API experiment met curl 2 (forms)

## Beschrijving
Dit experiment toont hoe je met CURL form-data verstuurt naar een REST API.
De POST-request gebruikt multipart/form-data, vergelijkbaar met een HTML webformulier.

## Gebruikte API
Test API:
https://httpbin.org

## Doel
- form-data naar en REST API versturen
- GET / POST methode gebruiken om data op te halen of aan te maken
- Webformulier (zonder HTML) gebruiken
