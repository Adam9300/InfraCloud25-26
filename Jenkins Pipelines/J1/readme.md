## 6.3.6 Lab - Build a CI-CD Pipeline Using Jenkins

## Beschrijving
In deze lab wordt een eenvoudige CI/CD pipeline gebouwd met Jenkins.
De pipeline automatiseert build-, test- en deploy-stappen.

## Bestanden
- Index.html + style.css - webpagina
- sample_app.py – Simpele applicatie voor de pipeline
- sample-app.sh - verschillende directories en pipeline aanmaken
- README.md – Uitleg van de pipeline


## Pipeline Stappen
- Code ophalen uit GitHub
- Build uitvoeren
- Tests uitvoeren
- Applicatie deployen

## Doel
Leren hoe:
- CI/CD pipelines werken
- Jenkins pipelines worden geconfigureerd
- Automatisatie software delivery versnelt
