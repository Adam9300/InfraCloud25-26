# Lab-experiment (slides)

## Beschrijving
Dit experiment richt zich op het verkennen en begrijpen van de basis van een Python-ontwikkelomgeving. In de lab wordt gewerkt met een virtuele machine (DEVASC VM) om inzicht te krijgen in de Python-installatie, het gebruik van PIP en het opzetten en beheren van Python virtual environments.

De nadruk ligt op het correct installeren en isoleren van Python-packages, zodat verschillende projecten onafhankelijk van elkaar kunnen worden ontwikkeld zonder conflicterende afhankelijkheden.
